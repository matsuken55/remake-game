# ならべてポン

ならべて、そろえて、消していく。4x4盤面に赤・青・黄・緑のサイコロ（数字1〜4）をドラッグ配置するブラウザパズルです。

## 基本ルール

- 「補充」でストックとして4個のサイコロを生成します。
- バッチ4個がすべて盤面で固定されるまで、新しい4個は補充されません。
- ストックの一部だけを盤面に置いて「確定」すると、その未固定サイコロだけを固定し、役判定と消去処理を行います。
- 未固定サイコロはPC/スマホのドラッグで自由に移動できます。固定済みサイコロは鍵表示になり移動できません。
- 役判定対象は横4列、縦4列、斜め2本、2x2ブロック9個、四隅1個の合計20グループです。

## ジョーカー

- 補充では出現しません。
- Normalは役得点が400点、Easyは280点貯まるごとに1個獲得します。
- 最大ストックは2個で、ジョーカー欄からドラッグして空きマスまたはロック済み通常サイコロの上に置けます。
- ジョーカー表示には `assets/dice/JK.png` を使用します。
- 役判定では任意の色・数字として扱うワイルドカードです。

## BGM

- ゲームBGMは `assets/audio/bgm.mp3` を参照します。
- `bgm.mp3` はCodexのPRには含めず、GitHubの `assets/audio/bgm.mp3` に手動アップロードしてください。
- Codex PRにはmp3などの音声・動画・zip等のバイナリファイルを含めないでください。
- 初期状態はBGM ONですが、ブラウザの自動再生制限に合わせてページ表示直後には再生しません。最初に「ゲーム開始」または「補充」「確定」を押したタイミングで再生を試行します。
- BGMは `loop = true` でエンドレス再生し、`ended` イベントでも先頭へ戻して再生を再試行します。
- BGM ON/OFFボタンの状態はlocalStorageに保存されます。
- `assets/audio/bgm.mp3` が未アップロード、404、読み込み失敗、再生失敗の場合でもゲームは止まりません。

## ゲームオーバーとランキング

「確定」で固定した後、消去が発生せず盤面に空きマスがなくなるとゲームオーバーです。名前とスコアは端末のlocalStorageにNormal / Easy別で上位15件まで保存されます。ハイスコアもlocalStorageに保存されます。

オンラインランキングはGoogle Apps Script APIをJSONP方式で呼び出します。API URLはコード内で以下に設定しています。

```text
https://script.google.com/macros/s/AKfycbxPW_cRfBrfc9t9MYXnit6e1M7oSp85bYvjEce-gGLmDOn2IHrHBbN9T28bzyCX-397GA/exec
```

- ランキング取得は `action=rankings&mode=normal|easy&callback=...` を送信します。
- ランキング登録は `action=submit&mode=normal|easy&name=NAME&score=SCORE&callback=...` を送信します。
- ゲームオーバー時に該当モードのランキング上位15位以内に入った場合だけ名前登録フォームを表示し、登録するとlocalStorageへ保存したうえでオンラインランキングへ送信します。
- API送信に失敗してもゲームは止めず、「オンラインランキング送信に失敗しました」と表示します。
- ランキング閲覧画面ではオンラインランキングの取得を試み、失敗時は端末内ランキングへフォールバックします。
- Normal / EasyランキングはAPIにもlocalStorageにも分けて保存・表示します。
- GitHubトークンや秘密情報はブラウザコードに入れません。Apps Script URL以外の秘密情報は不要です。

## 点数

- 同色同数字4個: 240（消去）
- 同色1-4: 200（消去）
- 4色同数字: 180（消去）
- 4色1-4: 160（消去）
- 色4個 / 数字4個: 40
- 色2色x2個 / 数字2種類x2個: 25
- 4色 / 数字1-4: 20
