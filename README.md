# Four Dice Grid

Four Dice Gridは、4x4盤面に4色×数字1〜4のサイコロを配置して役を作るオリジナルのブラウザパズルです。既存ゲームの名称、ロゴ、画像、音声、文章は使用していません。

## ルール

- 盤面は4x4です。
- サイコロは赤・青・黄・緑の4色と、数字1〜4の組み合わせです。
- 1ターンごとに4つのサイコロを生成します。
- プレイヤーは生成された4つすべてを空きマスに配置します。
- 4つすべてを配置した直後に役判定を行い、判定後に次のROLLが可能になります。
- ジョーカー（★）は任意の色・数字として扱うワイルドカードです。

## 判定グループ

役判定は合計20グループで行います。

- 横4列
- 縦4列
- 斜め2本
- 2x2ブロック9個
- 四隅1個

## 消える役

成立すると得点し、該当グループのマスが空きます。

- 同じ色・同じ数字が4つ
- 同じ色で数字1,2,3,4
- 4色すべて違い、数字が同じ
- 4色すべて違い、数字1,2,3,4

## 消えない得点役

成立すると得点しますが、サイコロは盤面に残ります。

- 色だけが4つ同じ
- 数字だけが4つ同じ
- 色が2色x2個
- 数字が2種類x2個
- 4色すべて違う
- 数字1,2,3,4

## 開発

```bash
npm test
npm start
```

`npm start`でローカルサーバーを起動し、ブラウザで`http://127.0.0.1:4173`を開いて表示確認できます。
